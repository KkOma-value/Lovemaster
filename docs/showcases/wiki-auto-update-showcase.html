新增架构组件

两个新增服务 + 两个 CLI 脚本，无缝集成现有管线

KnowledgeAutoApprovalJob

反馈驱动自动审批引擎。每 10 分钟扫描 pending_review 候选，聚合用户正向/负向反馈信号后自动决策。

@Scheduled(cron="0 */10 * * * *")

三条规则：≥3 正向反馈→批准 / >7d 无反馈→清理 / >14d→归档

WikiGraphSyncService

Wiki 内容变更后自动调用 graphify 同步知识图谱。30 秒防抖合并高频触发，120 秒超时保护。

requestSync("approve") / requestSync("reinforcement")

所有写入路径 (approve/reinforcement) 均自动触发

wiki-update.sh

CLI 统一更新入口。内置锁机制防并发、3 秒防抖、自动生成 index.md + CHANGELOG.md + .wiki-status.json。

bash scripts/wiki-update.sh "trigger-source"

支持 post-commit / claude-code-edit / manual 三种触发源

三条自动决策规则

完全由用户反馈信号驱动，零人工判断

自动入库

条件：正向反馈 ≥ 3 条
动作：自动 approve
写入：knowledge/wiki/topics/

自动清理

条件：创建 ≥ 7 天
+ 零正向反馈
动作：自动 reject

自动归档

条件：创建 ≥ 14 天
+ 仍为 pending_review
动作：标记 unknown_topic
