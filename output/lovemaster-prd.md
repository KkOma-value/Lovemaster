# PRD: Lovemaster 前端重构 — Dribbble 插画背景风格

## 1. 背景与目标

### 问题陈述
当前前端已完成第一轮视觉重构（科技粉 → 暖陶土），但仍以纯色/渐变为主视觉。用户提供了两张 Dribbble 浪漫插画参考图，希望：
1. **将这两张图片直接渲染为前端背景**
2. 前端整体风格对齐参考图的暖桃色 + 青绿色植物美学

### 参考图
| 图片 | 描述 | 用途 |
|------|------|------|
| ref_image_1 | "Sunflower Love" — 多场景恋人插画合集，倾斜卡片网格 | 首页背景 |
| ref_image_2 | "Couple on Bench" — 雨中长椅情侣，青绿大叶装饰 | 聊天页背景 |

### 目标
- **背景**: 两张 Dribbble 参考图分别作为首页和聊天页的全屏背景
- **色调**: 在现有暖陶土基础上引入**青绿色 (Teal)** 植物装饰色
- **UI 层**: 所有前景组件采用**玻璃态 (glassmorphism)**，让背景插画透出
- **兼容**: 零后端改动，保持现有组件结构和功能逻辑

---

## 2. 功能需求

### 2.1 背景图渲染
- 首页: `ref_image_1.png`（合集图）作为 `background-image`，`cover` + `fixed`
- 聊天页: `ref_image_2.png`（情侣图）作为 `background-image`，`cover` + `fixed`
- **消除留黑**: 强制 `#root` 和 `body` 宽高为 `100vw/100vh`，必要时调整 `background-size` 以裁剪图片自带的黑边信箱效果，确保全屏覆盖。
- 使用 CSS `background-attachment: fixed` 实现视差滚动效果
- 需要在 `body` 或页面容器级别设置，确保全覆盖

### 2.2 色彩系统增强
- **新增**: `--color-teal` 系列（青绿色，取自参考图植物叶片）
- **调整**: `--bg-base` 从 #FAF7F2 调整为更接近参考图的暖桃色
- **保留**: 陶土主色 #C47B5A 完全匹配参考图整体暖调

### 2.3 玻璃态 UI 升级
- 所有卡片/面板: `backdrop-filter: blur(20px)` + 半透明背景
- 侧边栏: 毛玻璃效果 + 细边框
- 聊天输入框: 玻璃态底色
- 消息气泡: 适度透明度让背景若隐若现

### 2.4 首页重构
- 移除纯色渐变 Hero Card 背景，改为在插画背景上的玻璃态卡片
- Bento Grid 布局保留
- 所有卡片增强玻璃态效果

### 2.5 聊天页视觉更新
- 背景换为参考图 2
- Header / ManusPanel 均保持玻璃态
- **侧边栏 (Sidebar)**：全面重构，参考 ChatGPT 官网布局结构。包含醒目的置顶“新对话”按钮、简洁的历史记录列表（移除不必要的边框线），增强专业度。
- **输入区精简**：删除输入框右侧现有的“回首页”按钮，保持聊天输入的纯粹性和视觉焦点。
- 消息气泡保持可读性（适度背景不透明度）

---

## 3. 非功能需求
- **性能**: 两张背景图优化至 < 500KB 各（当前 bg-lovers.png 约 2.4MB 需压缩）
- **兼容**: `backdrop-filter` 需加 `-webkit-` 前缀
- **无障碍**: 文字对比度 ≥ 4.5:1（在毛玻璃层上）
- **响应式**: 375/768/1024/1440px 四断点正常

---

## 4. 不做什么
- 不修改后端 API/SSE 协议
- 不引入新 npm 依赖
- 不重构 hooks/services
- 不改变路由结构
