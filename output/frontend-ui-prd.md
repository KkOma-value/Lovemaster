# 前端 UI 与动效优化 PRD

## 1. 项目概述
基于图一的现有页面，通过删减冗余信息、精简界面元素、引入交错动效和微交互，提升整体 UI 的高级感和用户体验，最终达到图二所展示的视觉标准。

## 2. 需求列表

### 2.1 首页文案与结构精简（对照图二还原）
- **左侧主卡片**：保留 "Love Master" 与简短介绍，移除过长的说明性文字。
- **右侧功能卡片（2 个）**：更新为“恋爱陪伴”、“恋爱教练”，并配以相应短文案。
- **底部特性卡片（4 个）**：更新为“情感分析”、“话术建议”、“隐私保护”、“即时响应”。

### 2.2 用户下拉框优化
- **背景**：当前用户下拉菜单（Dropdown）尺寸过大，不够精致。
- **需求**：适当缩小尺寸（优化宽高比、内边距、字号），使其在视觉上更自然，但必须**完全保留现有功能**（如账号设置、退出登录等）。

### 2.3 交互体验提升 (动效)
- **卡片悬浮响应（Hover 态）**：所有功能、特性卡片在鼠标悬停时，需平滑向上移动 `8px` 并加深阴影面积（利用 CSS transition 处理，动画曲线建议 ease-out）。
- **交错入场动画（Staggered Entrance）**：页面网格卡片首次加载时，依次加载，每一项延迟 `0.05` 秒。
- **初始状态**：透明度 0（opacity: 0），缩放 0.9 或更小（scale down）。
- **终态**：透明度 1，缩放 1（正常尺寸）。

## 3. 验收标准 (Acceptance Criteria)
1. 首页文案结构完全对齐图二内容。
2. 用户下拉菜单明显比优化前尺寸更紧凑，且功能点击无异常。
3. 卡片悬浮出现明确上移 `8px` 和阴影加深的视觉反馈，过渡自然无卡顿。
4. 页面刷新时，可以清晰看到卡片按照设定延迟（0.05s）依次由小变大、由透明变实的交错入场动画。
