# UI/UX 设计规范更新方案

## 1. 界面优化与重组
我们将针对 "Love Master" 的首页首屏展现进行降噪和聚焦。

### 1.1 主信息区改造
- **剔除信息**：删除“发聊天截图，我先帮你...”之类的复杂描述。
- **留存信息**：“你的专属恋爱助手” 作为 sub-title。

### 1.2 快捷卡片文案同步
右侧两个大型入口卡片：
1. **恋爱陪伴**：温柔陪你聊天，随时倾听你的心声
2. **恋爱教练**：分析聊天记录，给你最专业的建议

底部四个优势说明卡片：
1. **情感分析**：智能解读聊天内容，帮您理解对方的真实心意
2. **话术建议**：提供贴心的回复建议，让每次对话都恰到好处
3. **隐私保护**：所有对话内容安全处理，你的秘密只属于你
4. **即时响应**：快速生成建议，不让你在关键时刻等待

## 2. 微交互规范 (Micro-Interactions)
### 2.1 悬浮态响应 (Hover)
- **触发对象**：所有的主卡片与子卡片。
- **变换参数**：
  - `transform`: 向上平移（`translateY(-8px)`）
  - `box-shadow`: 阴影扩展并加深（例如：`box-shadow: 0 10px 25px -5px rgba(0, 0, 0, 0.1), 0 8px 10px -6px rgba(0, 0, 0, 0.1);`）
  - `transition`: `all 0.3s cubic-bezier(0.4, 0, 0.2, 1)`（平滑惯性曲线）。

### 2.2 入场动效 (Entry Animation)
- **触发机制**：页面首次挂载时触发。
- **基础状态**：`opacity: 0`, `transform: scale(0.95) translateY(10px)`。
- **展示状态**：`opacity: 1`, `transform: scale(1) translateY(0)`。
- **交错规律**：每个子元素增加 `+0.05s` 的延迟，按顺序（如由于配置为 Grid，可从上往下拉出渲染）。

## 3. UI 组件尺寸规范调整
### 下拉菜单 (Dropdown Selection)
- **尺寸控制**：最大宽度不宜超过 `200px` 或根据内容自适应。
- **圆角与边框**：延续当前风格（如 8px/12px border-radius，辅以非常轻微的光晕阴影），保留原有点击交互。
