## 2. 核心设计策略

### 2.1 背景渲染方案
**用户明确要求两张图片作为前端背景渲染出来。** 方案：
- **首页 (HomePage)**: 参考图 1（合集网格图）作为全页背景，`background-size: cover; background-position: center; background-attachment: fixed`
- **聊天页 (ChatPage)**: 参考图 2（长椅情侣图）作为背景，同样固定覆盖
- 两张图作为静态资源放入 `src/assets/illustrations/`
- 所有前景 UI 组件使用**玻璃态 (glassmorphism)**：半透明背景 + `backdrop-filter: blur` 让背景透出

### 2.2 色彩系统升级
当前系统已有的陶土/奶油色系基本匹配参考图，但需要新增关键色：
- **新增青绿色 (Teal)**: 参考图中大量的青绿色植物叶片是视觉标志
- **调整背景色**: 从 #FAF7F2 调整为更接近参考图的 #F5C9A8 暖桃色
- **保留陶土主色**: #C47B5A 与参考图的整体暖调完全一致

### 2.3 玻璃态 UI 层
在插画背景上叠加 UI 时使用毛玻璃效果：
```css
background: rgba(255, 253, 249, 0.75);
backdrop-filter: blur(20px);
-webkit-backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.3);
```

## 3. 同类产品参考

### 3.1 恋爱/社交类 App 设计趋势（2025-2026）
- **插画背景**: Bumble、Hinge 等约会 App 大量使用品牌插画作为入口背景
- **暖色调**: 粉色/桃色/珊瑚色是恋爱类产品的主流色，但差异化方向在于"插画手绘感"
- **玻璃态 (Glassmorphism)**: iOS/macOS 风格的毛玻璃效果已成为主流 UI 构建手法

### 3.2 差异化机会
- 使用**真实手绘风格插画**（非抽象几何）作为背景，增加"故事感"
- 青绿色植物元素区别于常见的粉色系恋爱产品
- 直接将 Dribbble 参考图渲染为背景，保证设计方向的高度还原

## 4. 当前前端改动评估

### 已有基础（可复用）
- CSS Custom Properties (Token) 架构 — 只需更新色值
- 组件结构（Bento Grid, Sidebar, ChatArea, ManusPanel）— 布局不变
- SVG 插画资源目录 — 已存在 `src/assets/illustrations/`
- 玻璃态效果 — `HomePage.module.css` 已部分使用 `backdrop-filter`

### 需要改动
- `global.css`: 添加青绿色 token、调整背景色、body 背景图逻辑按页面区分
- `HomePage`: 背景改为参考图 1 + 玻璃态卡片叠加
- `ChatPage` / `ChatArea`: 背景改为参考图 2
- 所有卡片/面板: 增强玻璃态效果让背景透出
- 引入两张 Dribbble PNG 图片作为资产

### 不需要改动
- 所有 hooks / services / API 逻辑
- React Router 路由结构
- ManusPanel 功能逻辑
- 后端代码
